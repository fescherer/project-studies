<a name="install"></a>

## 🚂 Getting Started 🚂

To get this project up and running locally on your pc, follow the steps:

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/ofelipescherer/project-studies.git
   ```
2. Go to project's folder

```sh
cd alura/courses-alura
```
